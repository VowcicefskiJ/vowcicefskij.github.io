Professional Self-Assessment

Welcome to my Computer Science capstone ePortfolio! This collection represents my growth and learning over three years in the SNHU Computer Science program. Through these projects, I've discovered my passion for intelligent systems and robotics while developing the technical skills needed for a successful career in computer science.

Throughout my journey in the Computer Science program, I have transformed from someone who could write basic functional code to a developer who understands how to build sophisticated, secure, and scalable applications. My coursework has taught me that good software isn't just about making things work—it's about creating maintainable, secure systems that can grow and adapt. This portfolio demonstrates the technical and professional skills I've gained, particularly in areas that align with my career interests in embedded systems, automation technology, and drone systems.

My Learning Journey and Skills Development

Working in Teams & Communication

Through my coursework, I learned how important it is to write code that other people can understand and work with. My MVVM architecture project taught me how to organize code so that team members can easily contribute without breaking existing functionality.

Technical Communication

I developed my ability to explain complex technical concepts through documentation, code comments, and presentations. Writing these project narratives helped me practice communicating my thought process and technical decisions clearly.

Problem-Solving with Algorithms

My understanding of algorithms grew significantly through implementing data structures like binary search trees and learning template programming in C++. I now approach problems by thinking about efficiency and choosing the right tools for each situation.

Modern Development Practices

I learned to use current technologies and frameworks like Jetpack Compose for Android, Firebase for cloud integration, and modern security practices. These skills prepared me for working in today's development environments.

Security-First Thinking

Through implementing password hashing and input validation, I developed an understanding of why security matters and how to build it into applications from the start, rather than adding it as an afterthought.

Technical Skills I've Developed

My coursework has given me hands-on experience with:

  • Full-Stack Development: MEAN stack applications and understanding how all the pieces work together
  • Data Structures & Algorithms: Learning when and how to use different approaches for optimal performance
  • Secure Programming: Writing code that protects user data and prevents common vulnerabilities
  • Mobile Development: Building Android apps using modern architecture patterns
  • Database Design: From basic SQL queries to implementing complex analytics systems

My Career Goals and Professional Readiness

Where I Want to Go: Intelligent Systems & Robotics

The skills I've developed through these projects directly support my career goals in robotics and autonomous systems:

  • Embedded Systems: My C++ experience with memory management and performance optimization prepares me for resource-constrained environments
  • Real-time Systems: Understanding algorithmic efficiency helps me write code that responds quickly enough for control systems
  • Connected Devices: My database and cloud integration experience supports IoT development
  • Safety-Critical Applications: My focus on security and testing practices is essential for autonomous systems where failures can be dangerous

How I'm Prepared for Professional Work

Through this program, I've learned skills that will help me contribute to development teams immediately:

  • Implementing security best practices from the beginning of projects

  • Working with databases to create applications that provide real business value

My Growth: This program taught me to think like a computer scientist—approaching problems systematically, considering multiple solutions, and always keeping the end user in mind. I'm excited to apply these skills in my career and continue learning as technology evolves.
